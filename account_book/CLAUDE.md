# 项目说明

这是一个安卓个人智能记账本 MVP。

技术栈：
- UI：Kivy / KivyMD
- 数据库：SQLite
- OCR：PaddleOCR
- LLM：先 mock，后续接 DeepSeek/豆包/通义千问
- 打包：Buildozer

核心功能：
1. 文字记账
2. 图片 OCR 记账
3. 账单保存到 SQLite
4. 日/周/月/年账单查询
5. 分类统计

不做：
1. 语音记账
2. 登录注册
3. 云同步
4. 多人账本

时间要求：
所有账单时间必须精确到分钟，格式为 YYYY-MM-DD HH:mm。

开发原则：
1. 每个 Agent 只改自己负责的模块
2. 不允许随意大面积重构
3. 改代码前先说明计划
4. 改完必须运行测试或给出手动测试方式
5. OCR 和 LLM 必须封装成 service
6. 数据库操作必须做好异常处理
